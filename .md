استخدم Encryption at Rest على مستوى البيانات والملفات. Laravel يدعم التشفير باستخدام AES-256/AES-128، والقيم المشفرة تكون موقعة بـ MAC لمنع التلاعب بها.

مثلاً في جدول patients لا تخزن الحقول الحساسة كنص عادي:

protected function casts(): array
{

        'chronic_diseases' => 'encrypted',
        'allergies' => 'encrypted',
        'medical_summary' => 'encrypted',
    ];
}

وللأدوية:

protected function casts(): array

2) Searchable Select في Filament يكتب أول 3 أحرف

في نموذج إضافة دواء للمريض:

Select::make('medication_master_id')
    ->label('اسم الدواء')
    ->searchable()
    ->getSearchResultsUsing(function (string $search): array {
        if (mb_strlen($search) < 3) {
            return [];
        }

        return MedicationMaster::query()
            ->where('is_active', true)
            ->where(function ($query) use ($search) {
                $query->where('brand_name', 'like', "%{$search}%")
                    ->orWhere('generic_name', 'like', "%{$search}%");
            })
            ->limit(20)
            ->get()
            ->mapWithKeys(function ($medication) {
                $label = trim(
                    ($medication->brand_name ?? '') .
                    ' - ' .
                    ($medication->generic_name ?? '') .
                    ' ' .
                    ($medication->strength ?? '')
                );

                return [$medication->id => $label];
            })
            ->toArray();
    })
    ->getOptionLabelUsing(function ($value): ?string {
        $medication = MedicationMaster::find($value);

        if (! $medication) {
            return null;
        }

        return trim(
            ($medication->brand_name ?? '') .
            ' - ' .
            ($medication->generic_name ?? '') .
            ' ' .
            ($medication->strength ?? '')
        );
    });

8) Prompt ممتاز لاستخراج الأدوية

بعد OCR، ترسل النص للـ AI بهذا الشكل:

استخرج من النص التالي البيانات الطبية فقط.
أرجع النتيجة بصيغة JSON فقط.
لا تضف تشخيصاً.
لا تضف معلومة غير موجودة في النص.
إذا كان النص غير واضح ضع confidence منخفض.

المطلوب:
- medications: name, dosage, frequency, duration, instructions, confidence
- conditions: name, confidence
- allergies: name, confidence
- lab_results: test_name, value, unit, reference_range, confidence
- notes: ملاحظات قصيرة عن النص غير الواضح

الخلاصة

نعم، نستخدم OCR وAI، لكن بالطريقة الآمنة:

المريض لا يكتب كثير
النظام يقترح
المريض أو الموظف يؤكد
الطبيب يرى ملخص مرتب
الأصل محفوظ دائماً
كل شيء حساس مشفر ومراقب بسجل وصول

Medication Master + Disease Master + OCR Review Screen
